태그에 클래스를 이용해서 색상을 바꿔주세요

ID / CLASS 의 이름 짓기 공식.

ID / CLASS 의 이름은 숫자로 지을 수 없다.

ID / CLASS 의 이름은 "-" 로 시작할수 없다.

ID / CLASS 의 이름은 직관적으로 지어야 한다.

ex) 상단의 메뉴를 만들때는 top~~ 로 짓는다.

ID / CLASS 의 이름이 2가지이상의 단어를 합칠경우는

언더바(_) , 대시(-) 낙타표기법(카멜케이스) 방식으로 만든다.

ex) 상단 메뉴를 뜻하는 top nav 를 지을때 방법

1 . top-nav 대시

2 . top_nav 언더바

3 . topNav 낙타표기법
